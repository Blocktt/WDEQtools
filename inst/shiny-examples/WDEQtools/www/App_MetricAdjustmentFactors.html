Instructions

This function allows the user to upload station information and assigns natural StreamCat factors. Some metrics are adjusted based on site-specific StreamCat estimates. See Background tab for metric calculations. Note, there are approximately 50 COMIDs in WY with some missing StreamCat data. Missing adjustment factor data will cause IBI scores to be incorrectly calculated. So, we recommend inputing an estimate from a nearby, similarly sized COMID. Elevation (ft) is also a metric adjustment factor, but elevation estimates are not provided in this tab, and so, must be provided by the user.

Follow the steps to the left to upload a site & COMID table. For more information on the derivation of these metric adjustment factors, please read the IDEM Diatom IBI report. Feel free to download an Example file to understand how the function works.
